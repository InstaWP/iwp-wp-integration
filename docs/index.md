# InstaWP Integration - User Guide

Welcome to the InstaWP Integration plugin documentation. This guide covers everything you need to set up, configure, and manage your InstaWP-powered site creation workflow.

## Quick Start

1. Install and activate the plugin
2. Go to **InstaWP > Settings** and enter your API key
3. Click **Refresh Snapshots** and **Refresh Plans** on the InstaWP Data tab
4. Edit a WooCommerce product and assign a snapshot in the **InstaWP** tab
5. Place a test order to verify everything works

For a full walkthrough, start with [Getting Started](getting-started.md).
